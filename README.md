Information:

Based off the legacy MilkShape 3D Driver PC DMODEL importer plugin by Soapy.

Ported to Blender 3.3+

Believe the original tools are from: https://drivermadness.net/forum/viewtopic.php?t=2075

Requires DMODEL to be extracted from `*.lev` of your choice.

Supports:

- Driver: You Are The Wheelman (Windows)
- DRIVER (iOS 2.2.1) .ipa.
